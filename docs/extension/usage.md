# Extension Usage

## Load extension in Firefox

1. Open about:debugging.
2. Select This Firefox.
3. Click Load Temporary Add-on.
4. Select src/manifest.json from this repository.

## Test workflow with cookie lab

1. Start the testing app over HTTPS.
2. Open app.localtest.me and api.localtest.me routes.
3. Apply scenarios from the testing UI.
4. Open the extension popup and switch output modes.
5. Compare output with expected scenario behavior documented in the testing-app pages.
